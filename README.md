# Stamp Management System

## Overview
A Java desktop application for managing stamp collections, allowing users to organize owned stamps and maintain a wishlist.

---

## Features
- User authentication (admin and standard users)
- Manage stamp collections by categories
- Wishlist management
- Add, edit, and delete items
- GUI with multiple views

---

## Tech Stack
- Java
- OOP principles
- GUI (Swing)
- CSV file storage

---

## Architecture
- Object-oriented design with classes and abstraction
- Role-based user system
- File-based data persistence

---

## Data Storage
- The application uses CSV files for data storage, allowing persistent data without requiring a database setup.
